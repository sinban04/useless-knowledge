## 한국·일본 식문화 = 감칠 시너지의 분자적 극대화

§4.5에서 본 **짠+감칠 시너지**는 인간이 가장 일관되게 맛있다고 평가하는 조합. 한국·일본 식문화는 천 년에 걸쳐 이 시너지를 분자적으로 극대화한 진화.

### Yamaguchi 시너지 (1967)

```
y = u + γ·u·v
```
- u = 글루타메이트 농도, v = 5'-IMP 또는 5'-GMP 농도
- γ = 시너지 계수 (IMP γ = 1218, GMP γ = 2800)

→ **MSG + IMP 1:1 혼합이 단순 합산보다 8배 이상 강한 감칠**.

### 한식의 5축 감칠 source

| 식품 | 감칠 분자 | 비율 |
|---|---|---|
| **다시마 (곤포)** | **글루타메이트** (free Glu ~3%) | 매우 높음 |
| **멸치** | **5'-IMP** + Glu | 높음 |
| **간장** | Glu (콩 단백질 발효) + Na | 매우 높음 |
| **된장** | Glu·peptide (콩 단백질 발효) + Na | 높음 |
| **표고버섯 (말린)** | **5'-GMP** | 매우 높음 (시너지 계수 최대) |

### 한국 식문화의 분자적 사례

1. **다시 + 멸치 + 다시마** — Glu × IMP 황금 시너지
2. **김치찌개** — 김치(lactic + Glu) + 멸치 + 된장 (Glu) → 감칠 + 신 + 짠 다층
3. **된장찌개** — Glu (된장) + Glu (소금) + 다시 다양한 source
4. **간장 양념 고기** — IMP (고기) + Glu (간장) → 시너지 + 짠
5. **김치** — 발효 lactic + 채소 Glu + 마늘·고추 + 소금 → 5 modality 균형
6. **불고기 양념** — 간장 + 설탕 + 마늘 + 배 → 짠+감칠+단+향의 통합

### 분자적 통찰

> **한식의 "깊은 맛"은 감칠 시너지를 다양한 분자 source로 다층화한 결과**.
>
> 단일 자극 (예: MSG만)보다 **여러 자연 source가 합쳐진 시너지**가 훨씬 풍부 — Glu(다시마) + IMP(멸치) + GMP(표고)의 3축 시너지가 한식 다시 국물의 분자적 정의.

### 일본의 우마미 문화

같은 시너지를 활용하지만 다른 source 조합:
- **Dashi (出汁)**: 곤포 + 가쓰오부시 = Glu + IMP
- **Miso (味噌)**: Glu peptide + 발효 향
- **Soy sauce**: 한국과 유사하지만 발효 시간·균종 다름

### 비교: 서구 식문화

서구에서는:
- 치즈 (긴 단백질 분해 → Glu)
- 토마토 (Glu)
- 발효 햄·소시지 (Glu)
- 안초비 (IMP)

→ **이탈리아 토마토 파스타 + 파르메산 치즈**는 정확히 토마토 Glu + 치즈 Glu의 시너지. **이탈리아의 "감칠" = 한국·일본과 같은 분자 패턴**.

### 응용: 요리 설계

감칠을 강화하려면:
1. **두 source 이상의 Glu** (다시마·간장·치즈·토마토 등)
2. **IMP 또는 GMP source 추가** (멸치·고기·표고)
3. **적정 짠 농도** (0.9~1.2%) — 시너지 + 보존
4. **lactic 발효식품 동반** → 신·감칠 다층

→ 라면 국물·국·찌개·스튜의 "맛있는 베이스"는 결국 위 4가지의 통합.

---

## 한 줄

> **한식의 분자적 핵심 = 짠+감칠 시너지의 다층화. 다시마 + 멸치 + 표고 + 간장·된장이 Glu·IMP·GMP·Na를 천 년 단위로 조합한 결과.**
